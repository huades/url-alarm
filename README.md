# 网页定时器 / Web Timer Tasks

[English README](README_EN.md)

一款轻量级 Chrome 扩展，用于 **单次定时打开网页**、**定时打开网页**、**每隔N小时打开网页**、**后台访问网址** 与 **刷新指定标签页**。

---

## ✨ 功能亮点
- **定时打开网页**：按“间隔天数”自动打开指定网页
- **单次定时打开网页**：指定日期与时间，到点打开网页后自动移除任务
- **每隔N小时打开**：按“间隔小时数”自动打开指定网页
- **后台访问**：按“间隔分钟”后台访问网址（后台弹窗自动关闭）
- **定时刷新**：按“间隔分钟”刷新指定地址前缀的标签页
- **任务可编辑**：支持修改网址/地址前缀、间隔天数/小时/分钟，以及日期与时间（open 与 hourly 任务均支持“日期 + 时间”分开编辑）
- **任务搜索**：支持按网址 / 类型 / 间隔实时筛选任务，搜索时自动展开有结果的分组，并在分组标题后显示匹配数量（如 `📅 每隔N天打开网页 (3)`）


- **主题切换**：浅色 / 深色 / 跟随系统
- **立即执行**：手动触发任务并更新下一次触发时间
- **导入/导出任务**：一键备份和恢复全部定时任务

## 📦 安装方式（开发者模式）
1. 打开 Chrome → 扩展程序 → 开启 **开发者模式**
2. 点击 **“加载已解压的扩展程序”**
3. 选择本项目根目录

## 🧭 使用指南
### 1) 单次定时打开网页
- 填写 **网址**、**打开日期** 与 **打开时间**
- 点击 **“添加”**
- 到达指定时间后会自动打开网页，并从任务列表中移除

### 2) 定时打开网页
- 填写 **网址** 与 **间隔天数**
- 点击 **“添加”**
- 任务创建后，可在列表中修改：
  - **间隔天数**
  - **日期**
  - **时间（几点几分）**

### 3) 每隔N小时打开网页
- 填写 **网址** 与 **间隔小时数**
- 点击 **“添加”**
- 任务创建后，可在列表中修改：
  - **间隔小时数**
  - **日期**
  - **时间（几点几分）**



### 4) 后台访问网址
- 填写 **网址** 与 **间隔分钟**
- 点击 **“添加”**
- 任务创建后，可修改 **间隔分钟**

### 5) 定时刷新标签页
- 填写 **标签页地址前缀** 与 **间隔分钟**
- 点击 **“添加”**
- 任务创建后，可修改 **间隔分钟**

### 6) 立即执行
- 点击 **▶️ 立即执行**
- 任务会立刻触发，并更新下一次触发时间

### 7) 搜索任务
- 在顶部搜索框输入关键词，可按 **网址 / 类型 / 间隔** 实时筛选
- 搜索时会自动展开有匹配结果的分组
- 分组标题会显示匹配数量，例如：`📅 每隔N天打开网页 (3)`
- 点击 `×` 可一键清空搜索并恢复默认折叠状态

## ⚙️ 行为说明
- **open 任务**：按间隔天数 + 指定时间触发
- **once 任务**：按指定日期时间触发一次，执行后自动删除
- **hourly 任务**：按间隔小时数自动打开网页
- **bg 任务**：后台弹窗访问，默认 10 秒后关闭
- **refresh 任务**：刷新匹配地址前缀的标签页

## 🛠️ 常见问题
### Q: 任务不触发？
- 请确认浏览器未禁用该扩展
- open 任务需要指定有效日期与时间

### Q: 刷新任务没有生效？
- 请确认已打开相关页面，且地址前缀匹配
